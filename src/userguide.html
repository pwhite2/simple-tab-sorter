User Guide ()

Thanks for taking the time to read this page. Doing so will, hopefully, save you time and improve your experience with this extension! Please refer back to this page after receiving updates because behavioral changes may be introduced as a result of any bug fixes or enhancement requests.

Simple Tab Sorter or Super Tab Sorter - which one's right for you?

Simple Tab Sorter has grown in complexity since its initial release, as a result of user feedback (which is greatly appreciated) and significant changes made to the Chrome browser and API (no so much). It's reached the point where it doesn't feel as "Simple" as I had intially intended and I'm having to say "No" to feature requests that conflict with my design goals of being "Simple" and lightweight.

I created Super Tab Sorter, a trustworthy alternative to Simple Tab Sorter for people who want more functionality and customization, even if that means additional complexity and resource consumption.

If you want something simple and light weight, that will remain that way, you'll probably be happier with Simple Tab Sorter. Simple Tab Sorter has entered "maintenance mode" and Super Tab Sorter is where I plan on adding functionality.

Tab Group Behavior (New in v0.4.0)

Simple Tab Sorter officially supports Chrome's Tab Groups as of v0.4.0. Tab Groups will be relocated to the left off all ungrouped tabs, in alphabetical title order, and their contents will be sorted using the same settings as ungrouped tabs.

    If you wish to override the tab group order, I suggest naming your tabs with a numeric prefix that indicates your desired sort order. e.g., "1. Shopping", "2. Learning", "3. Work", etc.

Planned removal of Sort By Custom (v0.5.0)

In my opinion, the introduction of Chrome's Tab Groups has lessoned the need for "Sort By Custom (SBC)". I think SBC has its place, for more demanding users, but I believe that users most likely to want SBC are more likely to use Super Tab Sorter. For that reason, Sort By Custom will remain a sort option in Super Tab Sorter and be removed from Simple Tab Sorter to minimize complexity and potential user confusion.

General Usage and Behavior

Regardless of the settings you choose, it's important to understand the following 2 things:

  1. Tabs will only be sorted when Simple Tab Sorter's icon is clicked.
  2. Tabs will only be sorted in the browser window where the icon is clicked.
With this in mind, you can safely experiment with different sort options, without reshuffling all the tabs in your main window, by opening several tabs in a new browser window and experimenting with the sort options in that window. Please note that extension settings are global, so any setting changes you save will be applied to all open windows when you click the Simple Tab Sorter icon later.

Setting Explanations

Sort By

You can sort by Title, URL, or Custom. Only Sort by Custom's behavior is affected by the "Group From" and "Preserve order within groups" settings. Sort by Title and Sort by URL ignore those settings and sort strictly by localized page Title or URL in ascending order. All Sort By options support the "Group suspended tabs together" setting, which groups all suspended tabs to the left side of your browser window.

Sort pinned tabs

Simple Tab Sorter ignores pinned tabs by default. If you check the box, pinned tabs will remain grouped together but will be sub-sorted by the same criteria used to sort the rest of your tabs.

Group suspended tabs together (applicable if using an extension derived from The Great Suspender)

Other tab sorting extensions that I've used have grouped all tabs suspended by The Great Suspender together, because The Great Suspender modifies the URL of suspended tabs and retains the original URL as a query parameter. I was a major fan of the original The Great Suspender (before the transfer of ownership), but have found that sorting behavior annoying, which was one of my primary reasons for creating Simple Tab Sorter.

By default, Simple Tab Sorter will not Group suspended tabs together - your suspended tabs will be sorted as if they were unsuspended. If you check the box, all suspended tabs will be shifted to the far left side of your browser and maintain their relative position to each other unless "Preserve order within groups" is deselected, in which case the suspended tabs will be subsorted by URL.

Tab suspender extension id (required if "Group suspended tabs together" is unchecked)
Look at the URL in your browser's address bar, for any suspended tab, and you should see something similar to the following...
chrome-extension://noogafoofpebimajpfpamcfhoaifemoa/suspended.html#ttl=Google&pos=0&uri=https://www.google.com/

The example above shows the URL for a suspended "https://www.google.com/" tab that was suspended by The Marvellous Suspender (the de facto The Great Suspender successor) extension. The highlighted portion between chrome-extension:// and /suspended.html is the "extension id" that you should copy/paste into the input field directly below "Group suspended tabs together" in the Settings page - don't forget to click Save!

Please note that The Marvellous Suspender's extension id will appear in the "Tab suspender extension id" field until you change and save the value in that field. This value can be safely ignored if you're not using The Marvellous Suspender or an extension with a compatible URL convention.

Group From (Sort by Custom only)

This setting is used to determine whether the first instance of a "tab group" should be identified in order from "Left to Right" or "Right to Left". This allows people to quickly, and easily, determine their desired "tab group" order by manually rearranging only a handful of tabs before clicking on the Simple Tab Sorter icon.

While this may seem unusual, at first, I encourage you to experiment with Sort by Custom and this setting for a while - I think it will quickly grow on you if you're not a fan from the start.

Preserve order within groups (Sort by Custom only)

The default behavior of this extension is to preserve the pre-sorted order of tabs within the sorted tab groups. I find this preferable to sorting everything by Title or URL, but included this as an option for others who prefer to have everything sorted alphabetically. If deselected, tabs will be subsorted by URL.
