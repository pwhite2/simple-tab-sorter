Warning/Disclaimer

I'd like to acknowledge that my initial inspiration for creating this extension was the lack of an available tab sorting extension that wouldn't group tabs together that were suspended by the original The Great Suspender (TGS) extension. Unfortunately, the author of the original TGS extension sold the extension to an untrustworthy buyer who made secretive changes to the extension, resulting in the security community strongly advocating that people immediately remove the extension with Google, months later (IIRC), removing the original TGS extension from the Chrome Web Store as well as automatically removing it from all Chrome browsers.

As a result, many people have forked the original TGS repo on github and some have submitted safe copies of the extension to the Chrome Web Store (e.g.,The Marvellous Suspender). Simple Tab Sorter v0.3.1 added support for these derived works as long as they don't change the URL format of suspended tabs, with the exception of the extension ID portion of the URL, which can be configured under Settings.

Please note that I do not recommend the use of any of these extensions, there's no guarantee that any of the current or future derived works are trustworthy. If you use them, you use them at your own risk.
